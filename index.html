About PeepoSMP

PeepoSMP is a modded survival multiplayer server ran off the modpack FTB Infinity Evolved. This server was made for the smaller Twitch community just to get on and make good content and have fun with others in the community.

WHITELISTED

The server is whitelisted to keep the server moderated and so streamers can feel safe streaming on the server.

NO LAG

The server is located in Canada and is also dedicated with 5GB of ram.

DEDICATED SERVER

The server is a fully build dedicated server from StrixServers that is able to run 24/7 without lag.

VARIETY OF COMMUNITYS

The server has a variety of communitys and accept more for a more unique experience.

PROXIMITY CHAT

The server uses the voice proximity mod "GlibysVC" so players can talk in game.

MODDED

The server runs off the modpack FTB Infinity Evolved.
